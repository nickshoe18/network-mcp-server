# HPE Networking MCP Server Setup

Setup guides and config files for connecting Claude Desktop to 
Juniper Mist and HPE Aruba Central using the hpe-networking-mcp 
unified MCP server.

## Prerequisites
- Docker Desktop
- Node.js 18+
- Claude Desktop
- HPE GreenLake API credentials
- Juniper Mist API token

## Setup
See the guides in the docs/ folder:
- `docs/HPE_Networking_MCP_macOS_v2.0.docx` — Apple Silicon Mac
- `docs/HPE_Networking_MCP_Windows11_v2.0.docx` — Windows 11

## MCP Server
This setup uses the community hpe-networking-mcp server:
https://github.com/nowireless4u/hpe-networking-mcp

## Claude Desktop Config
See `claude_desktop_config.example.json` for the correct 
configuration to use with the unified server.
